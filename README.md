# STOCK TRACKER

<p>Created by: Kiran Sangha </p>
<p>Built with: HTML, CSS, JavaScript, API</p>

## Description:
<p>This project provides the latest updates on stock prices and trends. The application integrates TradingView chart widgets to visualize real-time price data and technical indicators for analyzing stock performance. Users can easily access up-to-date information, including open, high, low, close prices, and trading volume by fetching data from external market APIs through a simple search interface</p>

## Purpose

<p> This project was built to gain hands-on experience working with real-time financial data and REST API integration. The goal was to make stock tracking accessible and user-friendly, allowing users to quickly analyze market trends and pricing data. </p>

## What I Learned
<p>- Working with REST APIs</p>
<p>- Asynchronous JavaScript (async/await)</p>
<p>- Data visualization for financial applications</p>
<p>- Designing user-friendly interfaces for complex data</p>
